# model editing

给模型的的某个知识进行修改，比如美国总统从拜登变成了川普。

reliability: 修改的知识需要生效；

generalization: 针对修改的知识作一定的变化；
paraphrase: 对原问题作改写，不改变意思；
reverse: 川普是谁？
portability: 美国总统的指责；

locality: 其他的问题不能被修改；

# model merging

将多个模型的能力进行合并，持续研究中。

通常训练模型需要很多资料，开源的模型，并没有把这些资料开源出来。

同一个基础模型的参数是A0，A0就是一个多维的向量。当训练完某个特定能力的模型参数是A1，A1-A0得到的一个新的向量叫做task vector。

有另一个模型A2，此时A2参数 + task vector，A2的模型也具有了A1的能力。

相加: 将多个模型的能力合并；
相减: 去除掉某个不需要的能力（比如说脏话）；
类比: 语音识别的时候，文本转语音 + 实际语音，特定领域的文本转语音，此时训练资料没有特定领域的实际语音，使用类比，可以让模型学会特定领域的ASR能力；

模型的参数量越大，model merging越容易成功，如果两个模型改动到同一个参数，merging就会失败。
